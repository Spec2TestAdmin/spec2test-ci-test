# ログイン画面仕様書

## 概要
ユーザーがメールアドレスとパスワードでログインする画面。

## 機能要件

### 正常系
- メールアドレスとパスワードを入力してログインボタンを押すとダッシュボードに遷移する
- パスワードは入力時にマスクされる

### 異常系
- メールアドレスが空の場合、「メールアドレスを入力してください」と表示
- パスワードが空の場合、「パスワードを入力してください」と表示
- 認証失敗時、「メールアドレスまたはパスワードが正しくありません」と表示
- 5回連続で認証失敗した場合、アカウントがロックされる

### 非機能要件
- ログイン処理は3秒以内に完了すること
- パスワードはSHA-256でハッシュ化して送信すること
